## Salesforce CPQ Comparison and Integration

Monetizenow offers similar capabilities to Salesforce CPQ but with easier setup for B2B quoting and billing. Use our integration to relay data between systems.

| Feature | Monetizenow | Salesforce CPQ |
|---------|-------------|----------------|
| Quote Builder | Guided templates, upsells | Configurator rules |
| CRM Sync | Real-time automation | Native Salesforce |
| Usage Metering | Built-in tracking | Add-ons required |
| Multi-currency | Native support | Standard |

<Steps>
  <Step title="Enable OAuth" icon="key">
    Generate an OAuth app in Salesforce and note your `client_id` and `client_secret`.
  </Step>
  <Step title="Configure in Dashboard" icon="settings">
    Navigate to Monetizenow Dashboard > Integrations > Salesforce. Enter your credentials.
  </Step>
  <Step title="Test Sync" icon="zap">
    Create a test quote and verify it appears in Salesforce.
  </Step>
</Steps>

<Callout kind="tip">
  Map custom fields during setup to ensure seamless data relay.
</Callout>

## Payment Gateway Setup

Connect Stripe or PayPal for multi-currency billing. Monetizenow handles refunds and credits automatically.

<Steps>
  <Step title="Add Gateway" icon="credit-card">
    Go to Dashboard > Billing > Payment Gateways. Select Stripe.
  </Step>
  <Step title="Enter Credentials">
    Provide your `publishable_key` and `secret_key`.
  </Step>
  <Step title="Enable Currencies">
    Toggle USD, EUR, GBP in settings.
  </Step>
</Steps>

```javascript
// Example: Create a payment intent
const stripe = require('stripe')(`sk_test_YOUR_SECRET_KEY`);

const paymentIntent = await stripe.paymentIntents.create({
  amount: 2000, // $20.00
  currency: 'eur',
  metadata: { subscription_id: 'sub_123' }
});
```

## Webhook Configuration

Set up webhooks for real-time events like quote approvals or billing changes. Your endpoint receives JSON payloads.

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const express = require('express');
  const app = express();
  app.use(express.raw({type: 'application/json'}));

  app.post('/webhook', (req, res) => {
    const event = req.body;
    if (event.type === 'quote.approved') {
      console.log(`Quote ${event.data.id} approved`);
    }
    res.json({received: true});
  });

  app.listen(3000);
  ```
  ```python
  from flask import Flask, request, jsonify

  app = Flask(__name__)

  @app.route('/webhook', methods=['POST'])
  def webhook():
      event = request.json
      if event['type'] == 'quote.approved':
          print(f"Quote {event['data']['id']} approved")
      return jsonify({'received': True})

  app.run(port=3000)
  ```
</CodeGroup>

## Custom API Integrations

For advanced needs, use our REST API at `https://api.example.com/v1`.

<Request tabs="cURL,JavaScript">
  ```bash
  curl -X POST https://api.example.com/v1/quotes \
    -H "Authorization: Bearer YOUR_TOKEN" \
    -H "Content-Type: application/json" \
    -d '{
      "customer_id": "cust_123",
      "items": [{"product_id": "prod_456", "quantity": 1}]
    }'
  ```
  ```javascript
  const response = await fetch('https://api.example.com/v1/quotes', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      customer_id: 'cust_123',
      items: [{ product_id: 'prod_456', quantity: 1 }]
    })
  });
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "id": "quote_789",
    "status": "draft",
    "total": 1500
  }
  ```
</Response>

<ResponseField name="id" field-type="string" required="true">
  Unique quote identifier.
</ResponseField>
