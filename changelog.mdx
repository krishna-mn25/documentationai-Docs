## Recent Updates

Monetizenow releases updates regularly to enhance your B2B commerce experience. Review the latest changes below to understand new features, improvements, bug fixes, and any breaking changes.

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

- Introduced Usage Metering dashboards with real-time analytics for tracking consumption-based billing.

- Resolved PIM lifecycle management issues when updating product bundles.

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched Revenue Lifecycle automation, allowing dynamic pricing ramps and upsell recommendations during quoting.
- Added advanced templates to Quote Builder for faster quote renewals and custom workflows.

## Breaking Changes

API endpoints for billing now require `version=2` header. Update your integrations:

<CodeGroup tabs="JavaScript,cURL">
````javascript
const response = await fetch('https://api.example.com/v1/billing/invoices', {
  headers: {
    'Authorization': `Bearer ${YOUR_API_KEY}`,
    'version': '2'  // Required for v2.0.0
  }
});
````

````bash
curl -X GET https://api.example.com/v1/billing/invoices \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "version: 2"
````
</CodeGroup>

## Migration Notes

Review your `/billing` configurations and test endpoints in staging.

</Update>
